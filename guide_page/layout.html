Header

기본 Type

공통 : 스크롤 내리면 헤더 비노출, 올리면 헤더 노출되게 적용함 : .wrapper 에 .header_hide class로 제어

메인 헤더 색 : .header 에 .scroll 클래스로.. 스크롤발생시 배경색 들어가게 적용

기본 헤더 (서브메인, 마이페이지-서비스이용현황 제외한 전체 페이지 적용)

U+비즈마켓 > 홈버튼, 가입상담신청(팝업오픈), 메뉴 버튼으로 구성

Sub Type (서브메인, 마이페이지 > 서비스이용현황에만 적용)

홈버튼, 서비스명, 메뉴 버튼으로 적용되어있음


Footer

공통

히스토리 : 각 서비스별 고객센터 번호가 상이하여 서브메인과용 footer가 따로 있었으나 고객센터 전화번호 통일되면서 footer 전체 통일함

footer에 기입된 고개센터 전화번호와 우측 플로팅 전화버튼의 전화번호 동일

top버튼은 footer 영역이 노출될때 올라옴.

CSS

기본규칙
테스트 용도를 제외한 운영 서비스에는 @import 방식은 사용하지 않음
폰트 단위 : rem 단위 사용
레이아웃 단위 : px, % 기본적으로 사용 (vmin, vh, vw 필요 시 유동적으로 사용)

Font

폰트 스타일
폰트명 NotoSans (웹폰트사용 - 경량화폰트 사용)
font-family: NotoSans,"Helvetica Neue","Apple SD Gothic Neo",sans-serif;
굵기- 400, 700으로 적용, font.css에서 웹폰트 관리
국문 : NotoSans / 영문,숫자 : Roboto (roboto class사용하면됨)
Font Size 1.5rem (15px적용) // px->rem으로 폰트사이즈 쉽게 변환하기 위해 html에 font-size: 62.5% 로 초기화
Font Color 기본 : #333 / 서브컨텐츠 : #666 / 설명컨텐츠 : #898989
메인 타이틀 : #000
Font Weight 타이틀 : 700 / 기본 : 400
